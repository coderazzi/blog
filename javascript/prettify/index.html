google-code-prettify Demo

This page is just a demo case for the javascript library google-code-prettify.

I have no relationship whatsover with the library or the author; I merely needed to apply the formatting to several code snapshots on this website and it was easier for me to use this page: paste the code and check (and use) the resulting html.

The version currenly used on this page is 1-Jun-2011

The code to prettify can be pasted on the text area below; then, press the prettify me button; if the uglify me button is used, the produced html simply escapes the html entities, so the code can be pasted directly on html pages.
